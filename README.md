# Cyber Matrix PFP Generator

Modern React + TypeScript app powered by Vite, Tailwind CSS, and shadcn-ui. Generates customizable cyberpunk-style profile pictures with canvas effects.

## Tech stack

- Vite
- React 18
- TypeScript
- Tailwind CSS
- shadcn-ui (Radix UI)

## Getting started

```sh
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Project structure

- `src/` – application code
  - `pages/` – routes
  - `components/` – UI and features (e.g., PFP generator)
  - `assets/` – images and media
- `public/` – static assets served at root

## Notes

- UI components are from shadcn-ui and Radix primitives.
- The dev server runs on port 8080; alias `@` maps to `src/`.
